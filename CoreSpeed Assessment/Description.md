# Polymath Documentation Assistant

**A Deno-powered AI agent that automates research and report generation.**

The project demonstrates a modern multilayered AI system that:

1. **Researches topics** on the web using the Firecrawl MCP server to find authoritative sources
2. **Synthesizes findings** using Claude LLM to create coherent, well-structured insights
3. **Generates reports** as professional Markdown documents saved to disk

**Core capabilities:**
- Real-time web search integration via Model Context Protocol (MCP)
- Intelligent information synthesis using Claude AI
- Automatic markdown report generation
- Interactive CLI prompts for user input

**Tech stack:**
- Deno (runtime)
- Zypher Agent Framework
- Anthropic Claude (LLM)
- Firecrawl MCP Server (web crawling)

**Use case:** Automate the process of researching a topic and producing a comprehensive, synthesized report in minutes instead of hours.
